Name that Function
Opposite
Hypotenuse

Legend

The side on top is the numerator in this division.
The side on the bottom is the denominator.

How are these functions defined?

Sine = Opp/Hyp
Cosine = Adj/Hyp
Tangent = Opp/Adj
Cosecant = Hyp/Opp
Secant = Hyp/Adj
Cotangent = Adj/Opp

Relations between functions

The main Trigonometric functions are Sine, Cosine, and Tangent.
On the unit circle, Sine represents Y values, and Cosine represents X values.

Tangent is Sine divided by Cosine.
Sine / Cosine = (Opp/Hyp) / (Adj/Hyp) = Opp/Hyp.

The other three functions are reciprocals of the first three.
Cosecant is 1/Sine = 1/(Opp/Hyp) = Hyp/Opp
Secant is 1/Cosine = 1/(Adj/Hyp) = Hyp/Adj
Cotangent is 1/Tangent = 1/(Opp/Adj) = Adj/Opp

Why learn this?

These functions become as useful as numbers.
Once we learn Trigonometry it shows up in nearly every subject we cover afterward.
In Calculus we use trigonometric functions to squeeze an unmeasurable value between two extremes when we learn the squeeze theorem.
